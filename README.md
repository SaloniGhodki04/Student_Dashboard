# Student Fees Performance Dashboard

## Project Overview
The Student Fees Performance Dashboard is a Power BI project developed to monitor and analyze student fee collection data. The dashboard provides a comprehensive view of total fees, paid fees, pending fees, course-wise collections, and area-wise student distribution. It helps educational institutions track financial performance and make informed decisions regarding fee management.

## Objectives
* Monitor overall fee collection performance.
* Track paid and pending fees.
* Analyze fee collection across different courses.
* Understand student distribution by area.
* Identify courses contributing the highest revenue.
* Support data-driven decision making.

## Features

### Key Performance Indicators (KPIs)
* Total Fees Collected
* Total Paid Fees
* Total Pending Fees

### Interactive Visualizations
* Course-wise Fee Collection Analysis
* Student Distribution by Area
* Collection Percentage by Course
* Detailed Student Fee Records
* Area and Course Filters for Dynamic Analysis

### Data Insights
* Identify high-performing courses.
* Track outstanding fee amounts.
* Compare fee collection across locations.
* Monitor student enrollment and payment trends.

## Tools & Technologies Used
* Power BI
* Data Modeling
* DAX
* Data Cleaning and Transformation
* Interactive Dashboard Design

## Dashboard Components
1. KPI Cards displaying Total Fees, Paid Fees, and Pending Fees.
2. Student Fee Details Table with course and payment information.
3. Course-wise Fee Collection Bar Chart.
4. Student Distribution by Area Visualization.
5. Collection Percentage by Course Pie Chart.
6. Interactive Slicers for Area and Course Filtering.

## Key Skills Demonstrated
* Data Analysis
* Data Visualization
* Dashboard Development
* Business Intelligence Reporting
* KPI Design
* Power BI Data Modeling
* DAX Calculations

## Conclusion
The Student Fees Performance Dashboard transforms raw fee data into meaningful insights through interactive visualizations and KPIs. It enables institutions to efficiently monitor fee collection performance, identify pending payments, and improve financial planning through data-driven decision making.

### Course Enrollment and Revenue Dashboard

**Course Enrollment and Revenue Dashboard** is a Power BI project designed to analyze course performance based on student enrollments and generated revenue. The dashboard provides insights into total revenue, average revenue per course, enrollment trends, course-wise revenue contribution, and monthly enrollment patterns through interactive visualizations.

### Key Features
* Total Revenue, Average Revenue per Course, and Total Enrollments KPIs.
* Course-wise Revenue Analysis using bar charts.
* Monthly Enrollment Trend Analysis.
* Course-wise Enrollment Distribution using treemap visualization.
* Detailed course performance table with enrollments and revenue.
* Interactive filtering and data exploration.

### Technologies Used
* Power BI
* DAX
* Data Modeling
* Data Visualization
* Data Cleaning & Transformation

### Key Insights
* Identifies the highest revenue-generating courses.
* Tracks enrollment trends across different months.
* Compares course popularity based on enrollments.
* Helps institutions make data-driven decisions for course planning and revenue optimization.

### Conclusion
This dashboard transforms educational data into actionable insights by combining enrollment and revenue analysis in a single interactive report. It supports better academic planning, performance monitoring, and strategic decision-making through visual analytics.

### Student Performance & Attendance Dashboard

**Student Performance & Attendance Dashboard** is a Power BI project designed to monitor and analyze student academic performance and attendance records. The dashboard provides a comprehensive view of student marks, attendance percentages, subject-wise performance, and attendance categories through interactive visualizations and KPIs.

### Key Features
* Student-wise Marks and Attendance Analysis.
* Average Marks KPI for overall academic performance tracking.
* Subject-wise Average Score Analysis using Treemap Visualization.
* Student Performance Comparison through interactive bar charts.
* Attendance Category Classification (Good, Average, Low).
* Attendance Distribution Analysis using donut charts.
* Detailed student performance table with marks and attendance percentages.

### Technologies Used
* Power BI
* DAX
* Data Modeling
* Data Visualization
* Data Cleaning & Transformation

### Key Insights
* Identifies top-performing and low-performing students.
* Tracks attendance percentages across students.
* Analyzes subject-wise academic performance.
* Highlights attendance categories to monitor student engagement.
* Supports data-driven academic monitoring and decision-making.

### Conclusion
This dashboard transforms student academic and attendance data into meaningful insights through interactive reports and visualizations. It helps educators and institutions monitor student progress, identify improvement areas, and make informed decisions to enhance overall academic performance.
